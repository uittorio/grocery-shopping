# Recipory

A recipe management app for families — browse recipes, plan weekly meals, and generate shopping lists.

## Getting Started

```bash
cd recipory
npm install
npm run dev       # http://localhost:5173
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Production build |
| `npm test` | Run tests (watch mode) |
| `npm run test:run:once` | Run tests once (CI) |

## Current Status

Walking skeleton (Increment 1) — hardcoded recipe displayed through full hexagonal architecture stack.
